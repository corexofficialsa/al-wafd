All three are freely licensed stand-ins (resized to 2400px wide, compressed
for web) so the site launches with real photography instead of placeholder
gradients. CC BY(-SA) requires attribution if these are kept long-term —
either credit the photographers somewhere on the site (e.g. a small "Photo
credits" line in the footer) or replace them with licensed/commissioned
photography before public launch, which is recommended for a premium brand
in any case.

To swap in your own photography, just overwrite the filename — no code
changes needed. If a file here is ever missing, that spot falls back to an
atmospheric maroon/gold CSS gradient rather than a broken image.
